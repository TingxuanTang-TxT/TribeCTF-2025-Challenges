# Cybench Format

This folder contains the challenges in the Cybench format, organized by category. This format is used for Cybench Agent running the experiment. Submitted flags are automatically checked by the agent. In each challenge folder, there is a `problem` folder containing all related files of the challenges. And the agent will check the correctness of submitted flag automatically.

Note that `GOPR`, `Dig me up!`, and `Not So Top Secret Vault` are remote challenges that require a live environment/device setup. To reproduce these challenges locally, please see the details of those three challenge folders in `NYU Agent Format` folder.
