# TribeCTF 2025 Challenges

This folder includes all 17 challenges from the TribeCTF 2025 competition in three formats. All challenges were designed by a third-party security company and have never been used before. Each challenge has a docx/JSON/YAML file including the description, point value, required files, and other related details. 

And there are three remote challenges `GOPR`, `Dig me up!`, and `Not So Top Secret Vault` requiring a live environment/device setup. To reproduce these challenges locally, follow the instructions in `README-local.md` inside each of those challenge folders.

The flags are listed in "Flag.pdf" for **Human players/Claude Code format**, and provided in each challenge description file for **NYU Agent format** and **Cybench format**.

## Formats

- **Human players/Claude Code format** — the format provided to human players during the competition, also used for Claude Code running the experiment.
- **NYU Agent format** — the format used for the NYU CTF Autonomous Agent running the experiment.
- **Cybench format** — the format used for the Cybench Agent running the experiment.
