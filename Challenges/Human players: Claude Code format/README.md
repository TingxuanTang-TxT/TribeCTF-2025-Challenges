# For Human Players and Claude Code

This folder contains the challenges in the format that was provided to human players during the competition. This format is also used for Claude Code experiment. For human players, submitted flags are checked automatically by the competition system. For Claude Code, submitted flags are manually checked by the researcher. The correct flag of each challenge can be found in 'Flag.pdf'.

In each challenge folder, the 'Challenge.docx' shows the name, category, description, and value of each challenge, and the rest files are all required to solve the challenge.

Note that `GOPR`, `Dig me up!`, and `Not So Top Secret Vault` are remote challenges that require a live environment/device setup. To reproduce these challenges locally, follow the instructions in `README-local.md` inside each of those challenge folders.
